# Platform Shift Notes

## Purpose

This file preserves working notes about the shift from old desktop-first computing toward app-led, mobile-first, browser-first, and Linux-capable systems.

These notes are part of Jet Notes for the American Communications Online archive.

## 2026-06-15 Note: Apps, Devices, and Operating Systems

The working view is that people are moving away from the old model where Windows desktops and laptops controlled most software use.

Today, apps on cell phones and tablets lead much of daily online behavior. Apple phones, Android phones, tablets, and browser-based tools shape how people communicate, publish, watch video, use maps, run marketing, and access AI.

## Platform Direction

Key points for the archive:

- apps now guide many users more than desktop software
- phones and tablets are primary devices for many people
- browser-based tools reduce dependence on old desktop installs
- some users and organizations may move away from Windows-only workflows
- Linux can run useful software on standard hardware
- Dell and Lenovo machines can support Linux-based work
- public archives should avoid depending on one corporation or one operating system

## Infrastructure Note

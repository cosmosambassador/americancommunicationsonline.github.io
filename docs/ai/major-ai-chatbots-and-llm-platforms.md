# Major AI Chatbots, LLM Platforms, and AI Assistants

**Research update dated June 12, 2026**  
**Prepared for the TJ Morris / ACIR public knowledge archive**

Artificial intelligence is no longer represented by one chatbot or one company. The current ecosystem includes public-facing assistants, underlying large language model families, web-research engines, workplace copilots, coding assistants, open-model communities, and tools that run models locally.

The distinctions matter:

- An **AI assistant** is the public application people use to ask questions, write, research, create images, or analyze files.
- A **large language model (LLM)** is the underlying model family that powers an assistant or developer service.
- A **multi-model platform** provides access to models from several companies.
- An **open-model ecosystem** distributes downloadable or adaptable models and related tools.
- A **specialist copilot** is designed primarily for coding, workplace productivity, or enterprise operations.

## Current Notes as of June 12, 2026

### DeepSeek

DeepSeek's official website currently promotes **DeepSeek-V4 Preview** as available through its web service, app, and API. DeepSeek also provides a public chat assistant for coding, content creation, file reading, and other general tasks.

### Mistral AI

Mistral AI has renamed **Le Chat** to **Mistral Vibe**. The company describes Vibe as a unified AI chat assistant and agent for work and code. Older articles may still refer to Le Chat.

### xAI and Grok

Grok is developed by **xAI**. It is available as a standalone assistant and includes web search, reasoning, coding, voice, image, video, and document-analysis capabilities. Grok is associated with X, formerly Twitter, but xAI is the developer.

### Meta AI and Llama

**Meta AI** is the public assistant. **Llama** is the model family. Meta AI should not be listed as if it were identical to the underlying model.

### Perplexity

Perplexity is best described as an **AI answer engine**, not as one independent LLM. It researches the open web and routes questions across multiple frontier models.

### Hugging Face and Ollama

Hugging Face is not a single chatbot. It is a major open-model and machine-learning platform. Ollama is a tool for running supported models locally and, where needed, using cloud resources.

## Why This Matters for Independent Media

Writers, researchers, publishers, podcasters, and community builders should not treat every AI system as interchangeable. Each platform has different strengths, business incentives, access rules, privacy policies, integrations, and publishing workflows.

The practical approach is to:

1. Keep the permanent archive on websites and GitHub-controlled files.
2. Use AI platforms as research, drafting, analysis, coding, and media-production tools.
3. Verify factual claims before publication.
4. Preserve source links and update dates.
5. Separate public information from private operational records.
